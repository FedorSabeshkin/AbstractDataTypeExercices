В языке Java концепция класса как модуля выражается несколькими средствами:
Одноименные modules введенные в java 11, как еще одно из средств для инкапсуляции и решения проблемы с адом зависимосте - поддержание автономности классов.
Пакеты - папки в которых хранятся классы и относительно которых может вестить отсчет для модификаторы доступа package private. 
Класс - как .java файл в котором содержится исходный код программы. Другие классы могут подключаться инструкцией import. 
Модификаторы доступа - public, private, protected, package private - которые позволяют нам задавать ограничения на то, на какие модули 
мы делаем доступными и в каких областях. Возможны собственные краевые случаи, которые решаются с помощью использования modules.
